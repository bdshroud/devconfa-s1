## Prompt 1 – Sponsors Section

I am a beginner learning HTML and CSS.

Help me recreate the Sponsors section from the provided design using only HTML5 and CSS3.

Requirements:
- Do not use Bootstrap, Tailwind, or any CSS framework.
- Use semantic HTML.
- Use Flexbox or CSS Grid where appropriate.
- Create clean, readable, beginner-friendly code.
- Use unique class names to avoid conflicts with other sections.
- Include:
  - Section title
  - Subtitle
  - Sponsor description
  - "Become a Sponsor" button with hover effect
  - Sponsor tiers (Platinum, Gold, Silver, Virtual)
  - Logo layout
  - Community Partners section
- Add comments in the HTML and CSS.
- Make the layout responsive.
- Explain why each HTML element and CSS property is used so I can learn.

## Prompt 2 – Learning Prompt

Act as my HTML & CSS mentor.

Do not simply generate the final code.

Instead:
1. Help me analyze the design.

4. Explain the purpose of each CSS property.
5. Guide me step by step.
6. Correct my mistakes instead of rewriting everything.
